C++ I/O
=======

.. role:: cpp(code)
   :language: cpp

All methods belongs to the :cpp:`#include<iostream>`.


Basic Input
-----------

Basic types:

.. code-block:: cpp

    int a; long int b; long long int c;
    float d; double e;
    char f;
    cin >> a >> b >> c >> d >> e >> f;

Get lines with spaces:

.. code-block:: cpp

    string line;
    getline(cin, line);



Basic Output
------------

.. code-block:: cpp

    cout << a << b << c << d << e << f << line << endl;
