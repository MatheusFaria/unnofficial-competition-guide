C Operators
===========

Comma `,` operator
------------------

This operator evaluates all expressions and discards all the returns, except
the last one

.. code-block:: cpp

    int i = (1, 2) // the value 2 will be assigned to i
    while(scanf("%d", &a), a); // Continues the loop while a != 0
